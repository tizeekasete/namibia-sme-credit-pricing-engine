# Namibia SME Credit Risk & Pricing Engine

## Overview
This project builds a data-driven credit risk and loan pricing system for Small and Medium Enterprises (SMEs) in Namibia.

The goal is to simulate how financial institutions assess borrower risk, approve loans, and determine interest rates based on risk profiles.

## Objectives
- Analyze SME borrower data
- Build a risk scoring model
- Segment borrowers into risk categories
- Design a pricing strategy based on risk
- Create a dashboard to visualize decisions

## Tools & Technologies
- SQL (data extraction and transformation)
- Python (analysis and modeling)
- Power BI / Tableau (dashboard)
- GitHub (version control)

## Project Structure
- `data/` → raw and processed datasets
- `sql/` → SQL queries and transformations
- `notebooks/` → analysis and modeling
- `dashboard/` → BI dashboards
- `app/` → optional deployment layer
- `docs/` → documentation
